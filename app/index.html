Music Playlist Creator

This app uses the Yahoo! Weather API and Spotify API. When you enter your zipcode, and click the button, a GET request is sent to the Yahoo! Weather API to get the weather for that zipcode. The app will then use description as keywords to send another GET request to the Spotify API to return results of songs titles that match that keyword. This app was built using the Angular JS javascript framework and Bootstrap responsive framework.
